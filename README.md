# AI Task Manager

A Full Stack Task Management application built using Angular, Node.js, Express.js, and MongoDB.

## Features

- Add new tasks
- View all tasks
- Update task status
- Delete tasks
- REST API integration
- MongoDB database

## Tech Stack

### Frontend
- Angular 21
- TypeScript
- HTML
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

### Clone the repository

```bash
git clone https://github.com/hannafarsin/AI-Task-Manager.git
```

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
ng serve
```

Visit:

http://localhost:4200

Backend:

http://localhost:3000

## API Endpoints

GET /tasks

POST /tasks

PUT /tasks/:id

DELETE /tasks/:id
